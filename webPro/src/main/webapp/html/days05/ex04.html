ex04.html-head 태그

        1.head 태그
            -웹페이지의 정보를 나타내는 태그
            -메타데이터(데이터에 대한 데이터)의 컨테이너
            -html 메타데이터? html 문서에 대한 데이터
            -화면 표시X
            -메타데이터는 일반적으로 문서 제목, 스타일
        
        2.head 태그의 자식 태그
        1)title : 문서 제목
        2)style : 문서 스타일 정보 정의
        3)script: 문서 스타일 정보 정의
        4)meta  : 
            ㄱ. meta charset="UTF-8" 웹 문서에서 사용된 인코딩(문자셋) 정의
            meta태그 종류 검색(***)
        5)link : css 외부 리소스(자원)과 연결하는 태그 
        6)base : 이 페이지의 모든 상대 주소에 대한 기본 주소 및 대상을 지정하는 태그
