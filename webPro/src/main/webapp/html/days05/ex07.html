        [html 코딩 규칙(가이드)]
        1.문서 유형을 선언할 때는 "DOCTYPE html" : 대문자 권장
        2.요소(태그) : 소문자 권장
        3.모든 요소가 종료태그가 필요한 것은 아니다. 
            meta, br ,hr ,img 등등 종료태그 필요 X 
        4.img src=""  =등호연산자 앞 뒤에 공백 사용 X 
        5.p태그는 반드시 종료태그를 코딩하자. 
        ***** 6.요청 URL(폴더명,파일명) 은 반드시 대소문자를 구분한다. *****
        7.파일확장자 : .html .htm 
        외부 css 파일 : .cs 
        외부 js 파일  : .js
