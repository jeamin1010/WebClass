ex05.html - 반응형 웹

        1. 모든 장치(PC,모바일,탭 등등..)에서 웹 페이지가 보기 좋게 만드는 것. 
        2. 다양한 화면 크기와 뷰포트에 맞게 자동으로 조정되게 하는 것. 
        3. 
        4. 반응형 웹 디자인
            1) 반응형 이미지 처리
            너비(width)속성 : % 처리
            2) 반응형 텍스트 처리 :vw   [v]iewport [w]idth 
            

            뷰 포트는 브라우저 창의 크기이다.
            1vw = 뷰포트 너비의 1%

            뷰포트(브라우저 창 크기)너비가 50cm 인 경우 1vw = 0.5cm
            뷰포트(브라우저 창 크기)너비가 100cm 인 경우 1vw = 1cm

            3)미디어 쿼리 @media 코딩.
            - 반응형 웹을 구현하기 위한 쿼리
            - 미디어 쿼리를 사용해서 텍스트와 이미지의 크기를 장치에 맞춰 조정.... 

            예) @media (max-width:800px){
                    nav,section{
                        width : 100%;
                        font-size :10vw;
                    }
                }
