   [ js 쿠키( Cookie ) ]
   1. 본인 컴퓨터(클라이언트)에 작은 텍스트파일로 저장되는 데이터를 쿠키라고 한다.
   2. 한라인의 문자열
       "쿠키명=쿠키값;쿠키명=쿠키값;name=admin;쿠키명=쿠키값;""    (20개 까지 저장)
   2-2.  문자열 파싱...     
       
       
    3. 브라우저 쿠키  사용 허용 여부 결정
    4.  js 쿠키 사용 방법
        "document.cookie 속성"을사용해서 쿠키값을 저장/ 읽기.
    5. 쿠키 형식    
     name="nameValue"; expires="expireDate"; path="pathHolders"; domain="domainName"; secure 
     쿠키명=쿠키값;_만료시점=날짜;_path=/;_도메인=localhost; 보안처리
