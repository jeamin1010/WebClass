ex04_04.html : js 배열

  1. forEach()
     배열의 각 요소 마다  한 번씩 함수를 호출해서 반복 처리.
     
  2. map()
     배열의 각 요소에 함수를 실행하여 새로운 배열을 생성하는 함수   
  
  3. filter()
      필터링된 요소만을 갖는 새로운 배열을 생성하는 함수   
      
  4. reduce()
       배열의 각 요소를 실행하는 함수에서 단일 값을 생성하는 함수    
       
   5. every()
       배열의 모든 요소가 테스트를 통과하는지 여부 확인하는 함수    
       
   6. some()
       배열의 일부 요소라도 테스트를 통과하면 true/false 반환하는 함수    
       
   7. indexOf()    / lastIndexOf()
       배열에서 요소의 값을 검색하고 검색한 요소의 인덱스값을 반환하는 함수  ( - 1 )
       
   8.  find()  / findIndex()
        요소값      요소 인덱스
        5                 1
        var m = [3,5,2,4,1];
        첫 번째 5보다 같거나 큰 수를 찾고 싶다.
        
        var num = m.find( function (elt, i, array){
           return elt >=5;
        } )
     
     9. from()     
             길이(length) 속성이 있는 객체 또는 반복 가능한 객체에서 -> 배열 반환하는 함수
             문자열
             배열
             등등   
             
      10. keys() 
          배열의 키(인덱스값)가 있는 iterator 객체를  반환하는 함수
          
          entries()    key                        +   value 한쌍
                        배열의 인덱스값                요소의 값
                        [ 0 , 3 ]
