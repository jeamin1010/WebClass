    1. jQuery 이란 ? 
        - jQuery는 빠르고 작고 기능이 풍부한 JavaScript 라이브러리입니다.
        - 다양한 브라우저에서 작동. 
            ( html, css, js 브라우저 종류/버젼에 따라 적용 X  )
        -    HTML 문서 순회 및 조작, 이벤트 처리, 애니메이션 및 Ajax와 
              같은 작업을 훨씬 더 간단하게 할 수 있는 API. 
         - 다양성과 확장성의 조합으로
              jQuery는 수백만 명의 사람들이 JavaScript를 작성하는 방식을 변경
