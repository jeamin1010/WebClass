ex02.html - css 웹 글꼴

	[css 웹 글꼴]
	클라이언트에 설치되지 않은 글꼴 -> 웹 글꼴
	자동으로 웹 서버에서 글꼴 다운로드 -> 클라이언트에서 글꼴 사용.
	
	[다양한 글꼴 형식]
	1.트루타입글꼴(TTF) - 1980년대 후반 Apple/MS 개발한 표준 글꼴 형식
	2.오픈타입글꼴(OTF) - TTF 형식 기반의 확장 가능한 글꼴 형식(MS등록 상표)
	3.웹 오픈 글꼴형식(WOFF) - 웹 페이지에서 사용하기 위한 글꼴 형식
	4.SGV글꼴
	5.임베디드 오픈 타입(EOT)
