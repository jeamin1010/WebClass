ex08.html- 뷰포트(viewport) , 미디어쿼리

	[뷰포트(viewport)]
	1. 웹 페이지에서 사용자가 볼 수 있는 영역
	2. 표시 영역은 장치(디바이스) 마다 화면의 크기가 다르다.
	3. 스마트폰, 테블릿,컴퓨터 화면 - 뷰 포트 X
	4. 여러 장치에서 브라우저를 사용해서 웹 페이지를 장치 화면에 맞게 보여주어야한다.
	5. 장치의 크기 배율에 맞게 웹 페이지의 크기 제어...
			- 뷰 포트 설정
	
	
		-width=device-width 장치의 화면 너비를 따라서 웹 페이지 너비 설정
		-initial-scale=1.0 		 브라우저에서 페이지를 처음 로드할 때 초기 확대/축소 수준 설정

	[미디어 쿼리]
	1. css2에 도입 @media 규칙
	2. 다양한 미디어 유형에 따라 서로 다른 스타일 적용 정의
		다양한 미디어 유형 ? 컴퓨터 화면용 , 프린터 용, 휴대폰 ,텔레비전 등등
	3. css3 = css2 미디어 유형 + 확장
	4. 미디어쿼리는 [무언가를 확인] 후 사용할 수 있다.
		1) 뷰포트의 너비,높이
		2) 장치의 너비,높이
		3) 방향(테블릿,휴대폰의 가로 또는 세로 모드)
		4) 등등
		
	5. 미디어 쿼리를 사용해서 반응형 웹 구현...
	
	6. 미디어 쿼리 구문 형식
			@media not|only 미디어 장치 유형 and (true/false 표현식){
				css코딩
			}
		예) @media only screen and(max-width:700px){
			css 코딩
		}
		
		
		@media (max-width:700px){
			css 코딩
		}
		ㄱ. 미디어 장치 유형 : screen,print,speech 등등
