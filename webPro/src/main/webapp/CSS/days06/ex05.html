ex05.html - 전환(transition)

	[전환(transition)]
	1)지정된 기간동안 속성 값을 원할하게 변경하는것.
			width:0px 		-> :hover 		-> width:180px
			예) transition : width 0.5s ease;
	2)transition 약식
			-transition-delay 속성 					:X
			-transition-duration 속성 				:0.5s
			-transition-property 속성 			:width
			-transition-time-function 속성 	:ease
	
	3) 전환 효과를 만들려면 2가지 설정.
			(1) 효과를 추가하려는 css 속성
			(2) 효과 지속 시간
				지속시간을 지정하지않으면 기본값은 0으로 적용된다.
