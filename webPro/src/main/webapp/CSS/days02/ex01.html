1. CSS 선택자(selector)
       1) 단순 선택자 - 태그명 < 클래스명  < ID명(숫자로 시작 X)
                               선택자 점수 높으면 우선 순위가 높다. ( 점수 계산 ) 
       2) 결합자 선택자 - div p,  div > p,       div + p,           div ~ p
                                     후손      직계자식     형제레벨			div 와 같은 단계의 모든 p
                                   특정 관계를 가진 요소    
       3) 유사 클래스 선택자 - 특정 상태를 기반으로 요소 선택
                                     p:hover{}
                                     :의사 클래스
       4) 유사 요소 선택자  - 요소의 지정된 부분에 스타일을 지정하는 데 사용. 
                                     ::의사요소
                                     요소의 내용  앞 , 뒤에 내용 삽입
                                     

                                        float:left
                                        

                                        

                                     
                                     
                                     #section::after{
                                        content:"";
                                        clear:both;
                                        display:table;
                                        
                                     }
                                     
                                     

       5) 속성 선택자  - [속성=속성값]
                                [type='button']             ->           :button(jquery selector)
                                [class*='col-']
                                [class|='col-']
                                [class^='col-']
                                [class$='col-']
                                [class~='col-']
