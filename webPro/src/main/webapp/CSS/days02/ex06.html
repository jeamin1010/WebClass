ex06.html - 여백( margin )

   1. 테두리 외부 - 마진(여백)  
      테두리 내부 - 패딩
   2. margin 속성
       margin-top
       margin-right
       margin-bottom
       margin-left
       margin
   3. 여백 속성값 
        1) auto - 브라우저가 여백을 자동 계산
        2) 길이 단위 - px, pt, cm 등등
        3) % - 요소를 포함하는 부모요소의 너비의  %
        4) 상속 -  부모 요소 ( 여백) -> 자식 요소 (여백)
