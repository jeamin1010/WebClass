	[position 속성 - 요소의 위치 지정 방법]
	1.static 		(정적,기본값) - top,bottom,left,right 속성을 설정하는 의미 X
	2.relative 	(상대좌표)	- 기준(부모요소)
	3.fixed 		(고정)
	4.absolute 	(절대좌표) - 가장 가까운 부모(조상) 요소를 기준으로 배치
	5.sticky 		(사전적의미 : 끈적끈적한) - 사용자의 스크롤 위치에 따라 배치
	
	
	- position 속성을 먼저 설정하지 않으면 t/b/l/r 속성은 적용되지않는다.
	- top,bottm,left,right 속성을 사용해서 배치
