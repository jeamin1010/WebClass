ex01.html - overflow

	[overflow 속성]
	-요소의 콘텐츠가 너무 커서 지정된 영역에 맞지 않을 때
	콘텐츠(내용)을 자를지 또는 스크롤바를 추가할지 여부를 결정하는 속성.
	-li,a 인라인모드 메뉴 : float:left  		overflow:auto + 다음요소...
	
	-auto : 자동 : 콘텐츠가 넘치는 경우에만 스크롤바 O
	-visible(기본값) : 영역 외부에 넘치는 콘텐츠가 출력(랜더링) 되어진다.
	-hidden : 콘텐츠가 넘치는 부분은 잘라버린다. 보이지 않는다.
	-scroll : 콘텐츠가 넘치는 여부에 상관없이 무조건 스크롤바 O 
	
	참고)overflow 속성은 높이 설정된 + 블럭모드에만 적용된다.
